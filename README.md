# URL Shortener

Небольшой сервис для сокращения ссылок на Go. 
Позволяет создавать короткие алиасы для длинных URL, выполнять редирект по алиасу и удалять ссылки.  
Использует SQLite для хранения данных, Basic Auth для защиты операций создания и удаления.

## Технологии

- **Go** (1.21+)
- **Chi** – маршрутизатор
- **SQLite** – база данных
- **slog** – структурированное логирование
- **godotenv** – загрузка переменных окружения из `.env`


# API Документация
## Сократить URL
## POST /url

Требуется Basic Auth (логин и пароль из .env).

Тело запроса (JSON):

{
  "url": "https://example.com/very/long/path?with=query",
  "alias": "custom"  // опционально, если не указан – генерируется автоматически
}
Статус-коды:

201 Created – успешное создание
400 Bad Request – некорректный запрос (например, отсутствует url)
409 Conflict – алиас уже существует
401 Unauthorized – неверные учетные данные

## Получить оригинальный URL и выполнить редирект
## GET /{alias}

Производит редирект на оригинальный URL, соответствующий переданному алиасу.

Пример:
```
curl -v http://localhost:8080/custom
```

Cтатус-коды:

302 Found – редирект на оригинальный URL
404 Not Found – алиас не найден

## Удалить сокращённую ссылку
## DELETE /url/{alias}

Требуется Basic Auth.

Пример:
```
curl -X DELETE -u admin:secret http://localhost:8080/url/custom
```
### Статус-коды:

200 OK – успешное удаление
401 Unauthorized – неверные учетные данные
404 Not Found – алиас не найден
