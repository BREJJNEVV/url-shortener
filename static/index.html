О проекте

Простой сервис для сокращения ссылок, написанный на Go.

Технологии:

  • Go 1.21+
  • Chi роутер
  • SQLite база данных
  • Basic Auth для защиты

API Endpoints:

  • POST /url - создать ссылку (auth)
  • GET /{alias} - перейти по ссылке
  • DELETE /url/{alias} - удалить ссылку (auth)
